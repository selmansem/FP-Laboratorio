# Actividades de Laboratorio de FP (Fundamentos de Programación)

> [!NOTE]
> Este repositorio contiene las actividades de laboratorio correspondientes a la asignatura de Fundamentos de Programación (FP) de primero de Ingeniería Informática en la Universidad de Granada (UGR) propuestos por Manuel Lozano.

> [!WARNING]
> Este repositorio aun está en construcción. Algunas actividades pueden no estar completas o contener errores.
> Si encuentras algún problema, no dudes en abrir un issue o enviar un pull request.

> [!CAUTION]
> Para evitar publicar cualquier archivo con información personal o sensible, asegúrate de añadir en el nombre del archivo alguna de las siguientes palabras clave (en mayúsculas o minúsculas): NOPUBLIC, SECRET, PRIVATE.

## Listado de Ejercicios

-   Ejercicio 1: Declaración de variables
-   Ejercicio 2: Resolver un sistema de ecuaciones de 2 incógnitas
-   Ejercicio 3: Algoritmo babilónico para calcular raíces cuadradas
-   Ejercicio 4: Cálculo de un salario final con incremento fijo
-   Ejercicio 5: Cálculo del área del círculo y la longitud de la circunferencia
-   Ejercicio 6: Cálculo de PI
-   Ejercicio 7: Variación porcentual entre dos valores
-   Ejercicio 8: Mostrar cifras de un número de tres cifras en líneas separadas
-   Ejercicio 9: Determinar si un numero es positivo, negativo o cero
-   Ejercicio 10: Factorial de un número entero positivo
-   Ejercicio 11: Potencia de un número
-   Ejercicio 12: Extraer los digitos de un numero
-   Ejercicio 13: Sucesión de Fibonacci
-   Ejercicio 14: Cálculo de Factorial y Potencia
-   Ejercicio 15: Números perfectos
-   Ejercicio 16: Dibujar un árbol de Navidad (ASCII Art)
-   Ejercicio 17: Dibujar una X de tamaño n (ASCII Art)
-   Ejercicio 18: Dibujar un triángulo de Floyd (ASCII Art)
-   Ejercicio 19: Array. Limpieza de duplicados
-   Ejercicio 20: Encuentra el máximo

-   Ejercicio teórico 1: Determinar par o impar
-   Ejercicio teórico 2: Determinar si se puede formar un triángulo y el tipo según la longitud de sus lados
